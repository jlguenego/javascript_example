  • Open devtool with F12
  • Read the code in the main.js file.
  • What does strict mode does ?
    • Sloppy mode vs Strict mode
    • Variable cannot be used if not declared.
    • arguments.callee and arguments.caller do not exist in strict mode.
    • arguments cannot be reassigned.
    • this is global object in sloppy mode, undefined in strict mode.
    • Read only properties cannot be reassigned.
    • eval() creates its own scope.
    • with is forbidden in strict mode.
    • Octal numbers forbidden in strict mode.
